# ai-engineer

Repository structure for storing reusable prompts and agent assets:

```text
.
├── prompts/
│   └── common/
├── agents/
│   ├── github-copilot/
│   │   ├── agents/
│   │   └── skills/
│   └── claude-code/
│       ├── agents/
│       └── skills/
└── README.md
```

## Directory purpose

- `prompts/common/`: shared prompt library used across tools.
- `agents/github-copilot/agents/`: GitHub Copilot agent definitions.
- `agents/github-copilot/skills/`: GitHub Copilot skill definitions.
- `agents/claude-code/agents/`: Claude Code agent definitions.
- `agents/claude-code/skills/`: Claude Code skill definitions.
